== Description ==
Ajouter la note globale et un récapitulatif de vos avis Opinion System avec le shortcode [opinion_system_note_avis] ou grâce à un widget.

Ceci est la version de démonstration du plugin, pour acheter la version complète du plugin rendez-vous sur la page https://www.amauryrambaud.fr/plugin-opinion-system-note-avis/
Dans la version complète du plugin il existe également 2 autres shortcodes / widgets pour afficher la liste des témoignages client directement sur votre site WordPress.
Il est également possible de filtrer les notes et les commentaires par numéro adhérent (company_id) et de rajouter une limite au nombre de commentaires à afficher.

Le plugin est responsive et existe en 2 couleurs : bleu et blanc.
